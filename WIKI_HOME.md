## Project Summary

**HealthPal** is a comprehensive medical platform that:
- Connects patients with qualified doctors for telemedicine consultations (chat, audio, video)
- Enables medical equipment and medicine inventory management
- Supports donation and sponsorship campaigns for patients needing financial help
- Facilitates NGO missions and volunteer coordination
- Provides mental health resources and support
- Includes real-time messaging and health alerts

## Tech Stack

- **Backend**: Node.js + Express.js
- **Database**: MySQL + Sequelize ORM
- **Authentication**: JWT (JSON Web Tokens)
- **Security**: Helmet, CORS, bcrypt, Rate Limiting
- **Payments**: Stripe Integration
- **External APIs**: WHO, Google Translate, OpenFDA, and more
- **API Documentation**: Swagger/OpenAPI

## Key Statistics

- **21 Controllers** managing different modules
- **21 Route Files** for API endpoints
- **27+ Database Tables** with proper relationships
- **50+ API Endpoints** fully documented
- **7 External APIs** integrated for rich functionality
- **Role-Based Access Control** (Patient, Doctor, NGO, Admin)
